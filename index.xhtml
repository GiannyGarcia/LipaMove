How ETA works

  • Uses distance from your chosen point (GPS, map center, or terminal) and a simple traffic model.
  • The band (e.g. 5–8 min) is a guide, not a guarantee — especially if data is stale.
  • With a live IoT feed it's tighter; simulation is best for demos.

What you see on the map

  • Orange pin — jeepney; blue — Batrasco-style unit.
  • Orange / blue route lines — estimated path per unit from data.xml.
  • Cyan line with dots — core terminal loop reference.
  • Small blue dot — you after Locate me.

Commuter carry list

  • Fan or umbrella — Lipa weather shifts quickly.
  • Fare coins ready — faster than digging while queued.
  • Use Priority unit (★ on the card) if you ride the same route often.
  • If unsure about ETA, check crowd bars and the Confidence chip.

Emergency & services

  • Philippines emergency hotline: 911 (where supported).
  • For urgent help at terminals, ask terminal guards / staff.
  • LipaMove is for ETA / tracking — not official operator dispatch.
